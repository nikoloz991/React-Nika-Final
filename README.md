# Mixly — Cocktail Finder

Mixly is a responsive React app for discovering cocktails — browse, search, filter, and save favourites, with dark/light themes and English + Georgian localization.
Built with Vite, React 19, and SCSS Modules, using data from [TheCocktailDB](https://www.thecocktaildb.com).
